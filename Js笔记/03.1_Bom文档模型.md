## Dom文档模型

- BOM 是 **Browser Object Model，浏览器对象模型**。**BOM 是为了控制浏览器的行为而出现的接口**。

  对于JavaScript：为了能够**让JavaScript能控制浏览器的行为**，**JavaScript就有了一套自己的BOM接口**。

- Document object model

- 浏览器的关闭、打开新的窗口、前进、后退、地址上的地址
